Messaging App
Project requires JavaFX to be installed or Run in IntelliJ 2023.3.3 or higher with JavaFX package

How to use
1. In the Server Folder find and run Server.java.
A white box with lines will be launched on the screen. From here all messages between the clients is recorded.
A server must be running before opening the clients since the messaging between the clients runs through the Server.

2. In the Client Folder find and run Client.java multiple times.
The Client app has a section for messaging other Client programs and for messaging to all Clinet programs.
Multiple running clients are needed to simulate messaging between each other.

3. For each Client app it will ask you for a user name before entering.
The username will be recorded to the server so it knows where to send messages.

4. For sending messages to other users, press the the button of the user you want to send to, enter a message into the message bar and press the send button.
The user message space will rememmber all messages that you have sent and have been sent to you

5. For group messages enter a message and press the send button
The message will be sent to all current users.
